Bootstrap

På den här sidan har jag använt mig av bootstrap. Jag har använt mig av CDN-länkarna och tyckte det var ett väldigt smidigt sätt med CDN än att ladda ner paket för att testa på ramverket. Det är ovant att göra all css i htmlfilen men det är mycket enkelt att använda.

Nackdelen med bootstrap som jag kommit på hittills är att det inte finns så många färger att välja på i den vanliga versionen. Som jag förstått det fungerar det om man använder sig av SASS, men inte vanlig CSS. Det andra är att man känner sig väldigt låst, anledningen är kanske för att man går från att göra sin CSS fritt. Men det är ett bra hjälpmedel
